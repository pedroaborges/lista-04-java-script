# Exercícios de JavaScript

## Exercício 1

Aprendi a:

* Usar `querySelector`
* Mudar texto com `textContent`
* Trabalhar com clique de botão

## Exercício 2

Aprendi a:

* Adicionar e remover classes com `classList.toggle`
* Mudar o estilo com CSS

## Exercício 3

Aprendi a:

* Criar elementos com `createElement`
* Adicionar na página com `appendChild`
* Pegar valor do input e validar

## Exercício 4

Aprendi a:

* Usar variável como contador
* Atualizar o valor na tela
* Controlar para não ficar negativo

## Exercício 5

Aprendi a:

* Usar `querySelectorAll`
* Percorrer vários elementos
* Destacar apenas o item clicado
